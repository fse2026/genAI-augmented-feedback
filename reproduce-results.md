# Reproduce results

## Setup
1) Create a Python environment (3.10+ recommended).
2) Install dependencies:
   `pip install -r 3-analysis/env/requirements.txt`

## Generate summary tables
Run:
`python 3-analysis/scripts/compute_tables.py`

Outputs are written to:
`3-analysis/outputs/`

## Notes
- This anonymized package includes derived datasets and non-identifying materials.
- Course-specific KB documents and raw PDFs/logs are withheld during double-blind review.
